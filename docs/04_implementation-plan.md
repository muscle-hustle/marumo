# marumo 実装計画

## 概要

本ドキュメントは、marumoアプリケーションの機能追加時の実装計画を管理します。

個別の実装計画は [`implementation-plans/`](./implementation-plans/) ディレクトリ配下に配置されています。

## 実装計画一覧

- [v1.0](./implementation-plans/v1.0.md) - 初回リリース（2025-12-01完了）
- [v1.1](./implementation-plans/v1.1.md) - モザイク・ぼかし強度調整機能（実装中）

## 機能追加時の実装計画

機能追加が必要になった場合は、`implementation-plans/` ディレクトリ配下に新しい実装計画ファイルを作成してください。

ファイル名の命名規則：
- バージョン番号: `v1.1.md`, `v1.2.md` など
- 機能名を含める場合: `v1.1-feature-name.md` など

### 実装計画の記載例

```markdown
# 機能追加実装計画: [機能名]

## 影響範囲
- [ ] 既存コンポーネント: 
- [ ] 既存サービス:
- [ ] 型定義:
- [ ] ドキュメント:

## 実装ステップ
1. [ ] 要件定義の更新
2. [ ] 技術仕様の更新
3. [ ] 型定義の追加
4. [ ] 実装
5. [ ] テスト
6. [ ] ドキュメント更新

## リスク
- [リスク1]
- [リスク2]
```
